# 用户管理

管理系统中的所有用户,包括用户信息编辑、订阅管理、余额管理等功能。

## 表格列说明

### 启用 (Enable)
开关按钮,控制用户账号是否可用。禁用后用户无法登录。

### ID
用户唯一标识ID。

### 用户名 (Username)
显示认证方式(EMAIL/GITHUB等)和认证标识(邮箱/用户名)。认证标识后有勾标记表示已验证。

### 余额 (Balance)
用户账户余额。

### 赠送金额 (Gift Amount)
系统赠送的金额。

### 佣金 (Commission)
用户通过邀请获得的佣金。

### 邀请码 (Invite Code)
用户的专属邀请码。

### 推荐人 (Referer)
邀请该用户注册的推荐人。

### 注册时间 (Created At)
用户注册时间。

## 表格操作

### 编辑 (Profile)
打开用户资料侧边抽屉,包含三个标签页:

**基础信息 (Basic Info)**:
- 启用状态
- 余额
- 赠送金额
- 佣金
- 邀请码
- 推荐人ID

**通知设置 (Notify Settings)**:
- 余额变动通知
- 登录通知
- 订阅通知
- 交易通知

**认证方式 (Auth Methods)**:
- 查看和管理用户登录方式
- 邮箱、GitHub等

### 订阅 (Subscription)
打开订阅管理侧边抽屉,查看和管理用户订阅:
- 订阅列表
- 订阅状态
- 到期时间
- 流量使用
- 操作订阅

### 删除
删除用户(需确认),不可恢复。

### 更多 (More)
下拉菜单,快速跳转到:
- 订单列表
- 登录日志
- 余额日志
- 佣金日志
- 赠送日志

## 筛选功能

### 订阅筛选
选择产品套餐,查看购买该套餐的用户。

### 关键词搜索
输入邮箱、用户名等搜索用户。

### 用户ID筛选
直接输入用户ID精确查找。

### 订阅ID筛选
输入订阅ID查找对应用户。

## 创建用户

点击"创建"按钮打开表单:

**必填字段**:
- 认证类型: EMAIL/GITHUB等
- 认证标识: 邮箱地址或用户名
- 密码: 用户登录密码

**可选字段**:
- 余额: 初始余额
- 推荐人ID: 绑定推荐关系

## 使用场景

### 场景一: 禁用违规用户

1. 找到目标用户
2. 关闭"启用"开关
3. 用户立即无法登录

### 场景二: 充值用户余额

1. 点击"编辑"按钮
2. 进入"基础信息"标签
3. 修改余额金额
4. 保存更改

### 场景三: 查看用户订阅

1. 点击"订阅"按钮
2. 查看用户所有订阅
3. 可操作订阅(续期、重置流量等)

### 场景四: 追踪用户行为

1. 点击"更多"按钮
2. 选择对应日志:
   - 登录日志: 查看登录记录
   - 余额日志: 查看余额变动
   - 佣金日志: 查看佣金收益
   - 赠送日志: 查看系统赠送

## 重要提示

1. **启用控制**: 禁用用户会立即踢出登录
2. **余额修改**: 直接修改余额,注意金额单位
3. **删除用户**: 操作不可恢复,删除前确认
4. **订阅管理**: 可以在用户订阅页面直接操作
5. **日志追踪**: 通过"更多"菜单快速查看用户各类日志
6. **推荐关系**: 创建用户时可指定推荐人
7. **批量操作**: 目前不支持批量操作
8. **邀请码**: 每个用户有唯一邀请码
