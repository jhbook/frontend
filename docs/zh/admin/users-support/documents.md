# 文档管理

文档管理模块用于创建和维护帮助文档、知识库,为用户提供自助服务支持。

## 功能概述

### 文档列表

查看和管理所有文档:

- **文档标题**: 文档名称
- **文档分类**: 所属分类
- **文档状态**: 草稿/已发布/已下线
- **作者**: 文档创建者
- **创建时间**: 文档创建日期
- **更新时间**: 最后修改时间
- **浏览量**: 文档查看次数
- **评价**: 用户评分和评价

### 创建文档

创建新的帮助文档:

#### 基本信息

- **文档标题**: 清晰明确的标题
- **文档分类**: 选择所属分类
- **文档标签**: 添加关键词标签
- **文档摘要**: 简短内容摘要

#### 文档内容

- **富文本编辑**: 所见即所得编辑器
- **Markdown支持**: 支持Markdown格式
- **代码高亮**: 代码块语法高亮
- **图片上传**: 插入图片和截图
- **视频嵌入**: 嵌入视频教程
- **表格支持**: 插入表格数据

#### 文档设置

- **访问权限**: 公开/仅会员/指定用户组
- **显示顺序**: 设置显示优先级
- **相关文档**: 关联相关文档
- **SEO设置**: 设置SEO关键词

### 编辑文档

修改已有文档:

- 更新文档内容
- 优化文档结构
- 添加新的信息
- 修正错误内容

::: tip 提示
文档支持版本控制,可查看历史版本并恢复。
:::

### 删除文档

删除不需要的文档:

- **软删除**: 下线但保留数据
- **硬删除**: 彻底删除文档
- 保留浏览记录用于统计

## 常见问题

### 文档无法保存

检查:

1. 网络连接是否正常
2. 是否有编辑权限
3. 内容是否过大
4. 浏览器是否兼容

### 搜索找不到文档

可能原因:

1. 文档未发布
2. 访问权限不足
3. 关键词不匹配
4. 搜索索引未更新

### 文档格式错乱

解决方法:

1. 使用标准格式
2. 清理复制格式
3. 使用模板
4. 检查HTML代码
