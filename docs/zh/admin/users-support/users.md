# 用户管理

用户管理模块用于管理系统中的所有用户账户,是系统运营的基础模块。

## 功能概述

### 用户列表

查看和管理所有用户:

- **用户信息**: 用户名、邮箱、手机号
- **用户组**: VIP/普通/试用等
- **账户状态**: 正常/已禁用/已过期
- **余额**: 账户余额
- **流量**: 已用流量/总流量
- **有效期**: 订阅到期时间
- **注册时间**: 账户注册日期
- **最后登录**: 最后登录时间

### 用户筛选

快速查找用户:

- **按状态**: 正常/禁用/过期
- **按用户组**: VIP/普通用户
- **按注册时间**: 时间范围筛选
- **按流量**: 按流量使用情况筛选
- **按设备**: 在线/离线用户
- **搜索**: 用户名/邮箱/手机号搜索

### 添加用户

手动创建新用户账户:

#### 基本信息

- **用户名**: 登录用户名(唯一)
- **邮箱**: 用户邮箱地址
- **手机号**: 联系电话(可选)
- **密码**: 初始密码

#### 订阅信息

- **用户组**: 选择用户组
- **流量配额**: 设置流量额度
- **有效期**: 设置到期时间
- **速度限制**: 设置速率上限
- **设备限制**: 同时在线设备数

#### 账户设置

- **余额**: 初始账户余额
- **积分**: 初始积分
- **推广码**: 自定义邀请码
- **备注**: 用户备注信息

### 编辑用户

修改用户信息和配置:

- **更新基本信息**: 修改邮箱、手机等
- **调整订阅**: 修改流量、有效期
- **余额操作**: 充值或扣减余额
- **重置密码**: 为用户重置密码
- **更改用户组**: 升级或降级用户组

### 删除用户

删除用户账户:

- **软删除**: 标记删除但保留数据
- **硬删除**: 彻底删除用户及相关数据
- **批量删除**: 批量删除多个用户

::: warning 警告
删除用户会清除其所有订阅和使用记录,请谨慎操作。
:::

## 用户操作

### 禁用/启用

控制用户账户状态:

- **禁用账户**: 暂停用户服务访问
- **启用账户**: 恢复用户正常使用
- **禁用原因**: 记录禁用理由
- **自动解封**: 设置自动解封时间

### 流量操作

管理用户流量:

- **增加流量**: 为用户添加流量
- **扣减流量**: 扣除用户流量
- **重置流量**: 重置流量到初始值
- **流量记录**: 查看流量变更历史

### 时长操作

管理用户订阅时长:

- **延长时间**: 延长订阅有效期
- **缩短时间**: 减少订阅时长
- **设置到期**: 设置具体到期日期
- **永久订阅**: 设为永不过期

### 余额操作

管理用户余额:

- **充值**: 为用户账户充值
- **扣款**: 从账户扣除余额
- **转账**: 用户间余额转移
- **余额日志**: 查看余额变动记录

## 用户详情

### 基本信息

查看用户完整信息:

- 账户基本资料
- 订阅状态和配额
- 余额和积分
- 推广信息

### 订阅信息

查看用户订阅详情:

- 当前订阅套餐
- 流量使用情况
- 剩余有效期
- 节点访问权限

### 使用统计

分析用户使用数据:

- **流量统计**: 每日/每周/每月流量
- **在线时长**: 使用时长统计
- **节点偏好**: 常用节点分析
- **设备统计**: 使用的设备数量

### 订单记录

查看用户购买记录:

- 历史订单列表
- 订单金额统计
- 购买频率分析
- 续费情况

### 登录记录

查看用户登录历史:

- 登录时间
- 登录IP
- 登录设备
- 登录地点

### 在线设备

查看用户在线设备:

- 设备类型和系统
- 连接的节点
- 在线时长
- 强制下线功能

## 用户分组

### 用户组管理

创建和管理用户组:

- **组名称**: 用户组显示名称
- **组权限**: 节点访问权限
- **流量配额**: 组默认流量
- **速度限制**: 组速率上限
- **其他限制**: 设备数、并发数等

### 批量分组

批量设置用户组:

- 选择多个用户
- 指定目标用户组
- 批量更新用户组
- 保留或重置配额

## 用户搜索

### 高级搜索

使用多条件搜索用户:

- **组合条件**: 多个条件组合搜索
- **范围搜索**: 流量、余额范围
- **时间范围**: 注册、到期时间
- **自定义字段**: 备注等字段搜索

### 保存搜索

保存常用搜索条件:

- 快速应用搜索条件
- 管理已保存的搜索
- 共享搜索条件

## 用户导出

导出用户数据:

### 导出选项

- **导出格式**: Excel/CSV
- **导出字段**: 自定义导出字段
- **数据范围**: 全部或筛选结果
- **数据脱敏**: 敏感信息处理

### 批量导入

批量导入用户:

- 下载导入模板
- 填写用户数据
- 上传Excel文件
- 验证并导入

## 用户通知

### 发送通知

向用户发送消息:

- **站内信**: 系统消息通知
- **邮件**: 发送邮件通知
- **短信**: 发送短信通知
- **批量通知**: 批量发送消息

### 通知模板

管理通知模板:

- 创建消息模板

- 多语言模板

## 用户分析

### 用户画像

分析用户特征:

- **基础属性**: 年龄、地区分布
- **行为特征**: 使用习惯、活跃度
- **消费特征**: 客单价、复购率
- **流失风险**: 流失预警分析

### 用户价值

评估用户价值:

- **RFM分析**: 最近购买、频率、金额
- **生命周期**: 用户生命周期阶段
- **贡献度**: 收入贡献排名
- **活跃度**: 使用活跃程度

## 用户安全

### 安全设置

增强用户账户安全:

- **强制2FA**: 要求两步验证
- **IP白名单**: 限制登录IP
- **设备管理**: 管理授权设备
- **异常检测**: 检测异常登录

### 风控措施

防范恶意用户:

- **行为监控**: 监控异常行为
- **风险评分**: 用户风险等级
- **自动封禁**: 触发规则自动禁用
- **黑名单**: 永久拉黑用户

## 最佳实践

- 定期审查用户账户状态
- 及时处理到期用户续费
- 关注高价值用户需求
- 分析流失用户原因
- 保护用户隐私数据
- 建立用户分级服务体系

## 常见问题

### 用户无法登录

检查以下几点:

1. 账户是否被禁用
2. 密码是否正确
3. 是否触发了安全限制
4. 系统服务是否正常

### 流量异常

可能的原因:

1. 统计服务故障
2. 节点上报异常
3. 用户设备问题
4. 恶意使用

### 批量操作失败

检查:

1. 选择的用户数量
2. 操作权限
3. 数据格式是否正确
4. 系统资源是否充足
